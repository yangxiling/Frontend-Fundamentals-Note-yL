## 事件（Vue和React对比，非常重要）

Vue中的event事件是原生事件（MouseEvent）。就是DOM事件。



1、React中的event事件不是原生，是合成事件--->SyntheticEvent。但是模拟出了 DOM 事件所有能力

2、SyntheticEvent里面的event.nativeEvent是原生事件对象

3、React所有事件，都被挂载到 document 上。

4、和   DOM  事件不一样，和 Vue事件也不一样。
